A little about Simon...

As Lead API Engineer and a Full-Stack Developer at GoSquared, Simon has helped scale GoSquared's real-time user analytics systems to handle billions of requests per month. When not working on highly available, scalable systems in Node.js; he can be found experimenting with new technologies from frontend development to ops systems.

Super-Scalable Redis for Fun and Profit!

Track 2

Redis is a notoriously fast, in-memory, Key/Value store widely used for building quick & efficient Web Applications; but it has always had one major flaw - Scalabilty. In past times, the only way to scale Redis was Sharding - a difficult, time-consuming process. In this talk we'll cover Redis; working up from the basics, to building super-scalable systems using Redis' new Clustering capabilities.
